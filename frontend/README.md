#### Для установки приложения:

```
cd frontend - перейти в директорию приложения из корневой папки проекта
npm i - установка зависимостей
npm run start - запуск проекта
npm run build - сборка проекта


при запуске локально к порту через слэш нужно добавить название  route:
 например, http://localhost:60447/ - main 
 http://localhost:60447/course -  страница курса
 http://localhost:60447/blog
 http://localhost:60447/notfound
 http://localhost:60447/allcourses
 
```
