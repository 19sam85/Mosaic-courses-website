# Mosaic Courses Website
Website for Mosaic Courses that utilizes React and Django. It allows users to view available courses, book them, and access various other features.

### Tech stack
- React 
- Django
- Django REST Framework
- PostgreSQL
- Docker
- nginx
- GitHub Actions
